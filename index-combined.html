Rhythm Sketch

  

An easy way to jot down and edit musical notes.

Welcome to Rhythm Sketch.

This is pre-alpha software. Which means there's some work to do.

So far:

  1. Graphical notes on grid
  2. Click-and-drag notes to adjust pitch and timing
  3. Grab a note by the left or right edge to lengthen or shorten the note

Next:

  1. Real-time recording with computer keyboard serving as piano keys
  2. Graphically add and delete notes on grid
  3. Audio playback

Keep checking back as we add new features to this project!
